# 功能描述

> 用于信息化部备案 / 公示 / 介绍材料引用（100–200 字）

---

专利检索智能体是一款面向审查员的 AI 辅助检索系统。用户上传待审专利文献后，系统自动解析技术主题、发明构思、申请人等关键字段，通过多模型（Kimi、智谱、通义千问、DeepSeek、秘塔、OpenAI 兼容接口）× 多检索策略的笛卡尔积并行执行互联网检索，自动去重汇总并由报告模型生成结构化 HTML 报告（含对比文献、出处链接、GB/T 7714 引用格式）。全流程基于 React Flow 实时可视化看板监控，支持 Word / Excel / Markdown 三格式导出，可显著提升审查员检索效率与查全查准率。
