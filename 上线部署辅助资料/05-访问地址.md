# 访问地址

> 用于信息化部备案 / 用户访问入口说明

---

## 1. 主访问入口

| 访问方式   | 地址（示例，请以实际部署为准）                      | 备注         |
| ------ | ------------------------------------ | ---------- |
| **内网** | `https://patent.internal.xxx.gov.cn` | 主入口，推荐访问方式 |
| **外网** | `https://vpn.xxx.gov.cn`             | 需先登录单位 VPN |

## 2. 服务端口清单

| 服务          | 端口         | 协议         | 对外暴露 | 说明               |
| ----------- | ---------- | ---------- | ---- | ---------------- |
| 前端（Web）     | 443 / 3000 | HTTPS      | ✅    | 用户主访问入口          |
| Worker 健康检查 | 3001       | HTTP       | ❌ 内网 | 运维监控使用，前端定时 ping |
| 数据库         | 5432       | PostgreSQL | ❌ 内网 | 仅前端与 Worker 可访问  |

## 3. 健康检查地址（仅运维使用）

```
# 前端存活检查
GET https://patent.internal.xxx.gov.cn/api/health

# Worker 健康检查（需内网访问）
GET http://worker.internal.xxx.gov.cn:3001/health
```

预期返回：`{"status":"ok","timestamp":"..."}`

## 4. 浏览器兼容性

- **推荐**：Chrome 100+、Edge 100+、Firefox 100+、Safari 15+
- **最低分辨率**：1280 × 800（看板与报告页适配至 1920 × 1080）
- **不支持**：IE 任意版本

## 5. 访问常见问题

| 现象          | 可能原因            | 解决方式             |
| ----------- | --------------- | ---------------- |
| 页面打开但白屏     | 浏览器缓存或 JS 报错    | Ctrl+F5 强制刷新     |
| 登录后跳转失败     | 浏览器禁用第三方 Cookie | 启用 Cookie 或加入白名单 |
| 上传文件失败      | 文件超过 50 MB      | 压缩或拆分文件          |
| 检索任务一直"等待中" | Worker 未启动      | 联系系统管理员          |
| 看板长时间不更新    | Realtime 订阅断开   | 刷新页面，系统会自动重连     |
