# 使用说明

> 用于信息化部备案 / 新用户上手指引（500 字以内）

---

## 1. 登录系统

在浏览器访问部署地址，使用单位分配的邮箱与密码登录；首次使用请联系系统管理员开通账号。

## 2. 上传专利文献

进入"新建检索"，上传待审专利文献（支持 PDF / Word / Excel / TXT，单文件不超过 50 MB）。上传后点击"开始解析"，系统自动调用解析模型提取技术主题、发明构思、申请人、主要技术方案步骤等字段。解析完成后，可在表单中校对或人工修正。

## 3. 配置检索参数

- **选择 AI 模型**：勾选参与检索的模型（默认含 Kimi、智谱、通义千问、DeepSeek、秘塔），并指定用于生成报告的"报告模型"。
- **选择检索策略**：勾选检索策略（默认含"追踪检索""发明构思检索""主要技术方案步骤检索"）。
- **设置检索上限**：单策略返回文献条数（默认 10）。
- **高级选项**：是否启用定时预约执行。

## 4. 启动检索

确认参数后点击"提交"，系统创建检索任务并进入消息队列；Worker 进程并行执行「模型 × 策略」笛卡尔积子任务。

## 5. 实时看板监控

跳转至"检索进度"页面，可基于 React Flow 看板实时观察每个子任务状态（等待 / 执行中 / 成功 / 失败 / 已重试）。

## 6. 查看与导出报告

检索完成后跳转至"报告"页面，可在线查阅结构化 HTML 报告（含对比文献清单、相关性评价、出处链接、GB/T 7714 引用格式）。报告支持一键导出为 Word / Excel / Markdown 三种格式，可直接用于审查文档归档。
